Welcome to Conventional Ownage website repository.

This is the project page for the Conventional Ownage website repository. This website is a simple portal to display player stats and server stats for the Multi-1v1 servers, and other game servers that Conventional Ownage runs.

Built With

Node.js, Express, MySql, with EJS templating. All the data is processed server-side.
